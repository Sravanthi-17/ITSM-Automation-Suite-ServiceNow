# ITSM Automation Suite

## Project Overview

The ITSM Automation Suite is a custom ServiceNow application developed using App Engine Studio to automate IT issue management. The application streamlines issue creation, categorization, approvals, SLA tracking, notifications, and reporting.

---

## Features

- Custom IT Issue table
- Record Producer
- Service Portal
- Flow Designer Automation
- Auto Categorization
- Assignment Group Automation
- Business Rule for Priority Calculation
- Email Notifications
- SLA Definition
- Critical Issue Approval Workflow
- Reports
- Dashboard

---

## Technologies Used

- ServiceNow Zurich
- App Engine Studio
- Flow Designer
- Business Rules
- Record Producer
- Service Portal
- Notifications
- SLA
- Reports & Dashboards

---

## Workflow

```
User submits IT Issue
        │
        ▼
Record Producer
        │
        ▼
Auto Categorization
        │
        ▼
Priority Calculation
        │
        ▼
Assignment Group
        │
        ▼
If Priority = Critical
        │
        ▼
Approval Request
        │
        ▼
Manager/Admin Approval
        │
        ▼
State → Work in Progress
```
